     FATIAMENTO

frase[n] - mostra o caracter no numero de posição escolhido

frase[n:m] - mostra os caracteres dentre as posições escolhidas

frase[n:m:k] - mostra os caracteres dentre as posições escolhidas e pulando um número k de posições em ciclos

frase[:n] - mostra os caracteres anteriores ao caracter na posição n

frase[n:] - mostra os caracteres posteriores ao caracter na posição n

frase[n::k] - mistura de frase[n:m:k] e frase[n:]


     ANÁLISE

len(frase) - retorna um valor inteiro com o número de caracteres em uma string

frase.count('x') - retorna um valor inteiro com o número de vezes que a string x se repete em frase

frase.count('x', n, m) - retorna um valor inteiro com o número de vezes que a string x se repete na posição n a m de frase

frase.find('x') - retorna um valor que mostra em que posição de frase a string x está 

'x' in frase - Retorna com verdadeiro ou falso se a string x está presente ou não

    TRANSFORMAÇÃO

frase.replace('x', 'y') - Substitui uma string x por uma string y em frase

frase.upper() - Transforma todas as letras minúsculas de frase em maiúsculas 

frase.lower() - Transforma todas as letras maiúsculas de frase em minúsculas 

frase.capitalize() - Transforma todas as letras de frase em minúsculas e põe a primeira letra de frase em maiúscula

frase.title() - Transforma todas as letras procedentes de um espaço em maiúsculas

frase.strip() - Apaga todos os espaços inúteis de frase

frase.rstrip() - Apaga todos os espaços inúteis à direita de frase

frase.lstrip() - Apaga todos os espaços inúteis à esquerda de frase

    DIVISÃO	

frase.split() - Divide todas as palavras em strings únicas e são reorganizadas em uma lista

    JUNÇÃO		

'x'.join(frase) - Junta uma lista e adiciona o(s) caracter(es) entre as palavras
